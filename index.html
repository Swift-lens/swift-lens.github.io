FAQ

Common questions

Is SwiftLens really invisible to screen recorders?
Yes. SwiftLens uses the Windows SetWindowDisplayAffinity API with the WDA_EXCLUDEFROMCAPTURE flag. This is the same technique used by banking apps and DRM software. Zoom, Teams, OBS, and all standard screen recorders cannot capture SwiftLens windows.
What happens if I lose my internet connection?
SwiftLens automatically switches to Offline AI mode powered by the Phi-3 Mini model running locally on your CPU. You can continue getting answers without any internet connection until your subscription expiry date.
Can I use my license on multiple computers?
Each license key is for a single user. You can reinstall on the same machine as many times as needed. If you need multiple seats, please contact support.
What Windows versions are supported?
SwiftLens supports Windows 10 (version 1903 or later) and Windows 11. No GPU is required — it runs entirely on your CPU. Minimum 4 GB RAM recommended, 8 GB for offline AI mode.
